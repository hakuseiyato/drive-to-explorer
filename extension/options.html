Drive to Explorer - 設定

Drive for desktop がマウントされているローカルのルートパスを 1 つだけ指定します。

例: M:\ や G:\。Drive 階層 (マイドライブ / 共有ドライブ / フォルダ名…) が、このルート配下にそのままミラーされていることが前提です。

Drive REST API (OAuth) — 任意・推奨

OAuth Client ID を登録すると、DOM 解析より遥かに信頼性の高い API 経由で Drive 階層を解決します。
パス取得時の パスを表示 popup フラッシュも不要になります。
詳細手順: README の「OAuth セットアップ」セクション参照。

これを Google Cloud の OAuth クライアントに「承認済みリダイレクト URI」として登録
